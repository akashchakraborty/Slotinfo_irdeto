Hey there,Irdetee !


Welcome to the Slot information page of Irdeto Delhi Office


So, what will this website do ?

I am sure each one of has faced issues about not knowing which slot is in use or who is using a slot for long runs, log tests and what not. Sometimes many problems have also come up for this "not knowing" stuff. So here is a more convinient solution to your questions and it is a ot easier than checking Storm/Witbe everytime you wanna know about a slot


It will answer the following question...

Who has requested for the slot in question?

For which project the slot is being used ?

For how long will a slot be used for some project ?

Are there any Emergency Slot Request ?


Now, you might be thinking how to use it, right ?

We got you covered and here is brief on how to use it ...

Adding a new project

First, things first...see that "Options" in the top left ?
Yes, that one.
To create a new project go to that and select Add Project


Slot Reserve Request

Go to the Rack you wish by clicking on the"Rack" in the top.
Wait for the rack page to load.
Now you can see the Reserve button. Go click it and fill in the details.


Release Slot

Go to the Rack you wish by clicking on the"Rack" in the top.
Wait for the rack page to load.
Now you can see the Release button. Go click it and fill in the details.
